# Tesla Charge Companion 2.5

- 271 Superchargeurs Tesla uniques importés depuis `suc.xlsx` (mise à jour du 11/07/2026).
- Tarifs Tesla membre convertis en créneaux horaires.
- Coordonnées, adresses, puissance, nombre de stalles et liens Tesla intégrés.
- L’onglet Comparer affiche uniquement les 10 Superchargeurs les plus proches de l’adresse de départ.
- Classement par distance routière OSRM quand disponible, sinon par distance à vol d’oiseau.
- Les bornes AC locales restent disponibles dans l’onglet Bornes mais ne polluent pas la comparaison des Superchargeurs.
- Règle particulière conservée pour Grand Frais Les Clayes.
